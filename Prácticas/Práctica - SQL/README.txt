EJECUTAR NUEVAMENTE LA BD
1. Abrir pgAdmin (usamos pgAdmin III).
2. Conectarse a pgAdmin (localhost).
3. En la sección de arriba seleccionamos el ícono que dice "Create a new object of the same type as de selected object"
3. Nos aparecerá una ventana para crear una nueva base datos.
4. Agregamos el nombre a la BD, que en este caso puede ser RNRPP y seleccionamos aceptar. 
5. Seleccionamos la BD creada, y una vez ahí, nos dirigimos a la parte superior izquierda y seleccionamos el icóno que dice SQL.
5. Nos abrirá otra ventana, que es el SQL Editor. 
6. Seleccionamos el ícono de "Open file".
7. En nuestro ordenador buscamos el archivo "practica08.sql" y lo seleccionamos.
8. Seleccionamos todo el contenido del archivo y en la parte superior está el icóno de ejecutar: "Execute query".
9. Si en la ventana inferior nos aparece el mensaje "Query returned successfully with no result in # msec." implica que la base se pudo ejecutar con éxito. 
10. Abrimos la ventana de pgAdmin y refrescamos la base creada (RNRPP).
11. Abrimos el DDL y lo cargamos, de manera que se harán las politicas de mantenimiento.

EJECUTAR PARA POBLAR
12. Abrimos poblacion.sql y lo cargamos.

EJECUTAR CONSULTAS
13. Abrimos consutas.sql y cargamos cada consulta.
